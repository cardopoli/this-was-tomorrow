Quick reference for Markdown and Hugo patterns that work in TWT's body fields.

Markdown (works in body_intro / body_middle / body_after)
# Headingh1 (don't use - reserved for page title)
## Headingh2 - section heading with rule
### Headingh3 - sub heading in muted mono
**bold**bold text
*italic*italic text
[text](url)link
![alt](url)image (use Dropbox raw URL)
- itemunordered list
1. itemordered list
> textblockquote
---horizontal rule
blank linenew paragraph
Raw HTML (works because unsafe = true)
<img src="..." style="width:100%">Full-width image
<div class="audio-track" data-twt-bg="...">Custom audio player
<div style="display:grid;grid-template-columns:1fr 1fr;gap:4px">Two-column image layout
Hugo shortcodes (processed before markdown)
{{< audio-playlist url="..." bg="..." fg="..." >}}Folder playlist
Script embeds - DON'T work in body fields
<script>...</script>Stripped by markdownify even with unsafe=true. Use shortcodes instead.
